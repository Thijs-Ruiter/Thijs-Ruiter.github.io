Project Veilige Samenleving

Dit is het eerste project dat ik heb gedaan. In dit project gingen we een game maken over een veilige samenleving.

In deze game is het de bedoeling om rond te lopen en mensen te helpen met simpele klusjes. Door dit te doen gaat je reputatie omhoog in de game, waarmee je steeds een betere baan krijgt.

De game is gemaakt in Unity en het was mijn taak om code te schijven voor de klusjes en ze in de game te plaatsen. Daarnaast had ik ook nog een paar andere kleine taakjes die met c# te maken hadden
